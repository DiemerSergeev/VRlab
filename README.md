VR Laboratory: 
Hardware-Integrated SimulationAn immersive virtual reality training environment built with Unreal Engine, designed to provide high-fidelity simulations synchronized with real-world industrial or scientific equipment.
🌟 Overview
This project serves as a "Digital Twin" bridge. It allows users to interact with virtual representations of complex machinery while receiving real-time data or sending commands to actual physical hardware. It is designed for safety training, remote equipment control, and educational purposes.
🚀 Key Features
Real-time Hardware Synchronization: Bi-directional communication between Unreal Engine and physical controllers (PLC/Arduino/Sensors).High-Fidelity Visuals: Photorealistic 3D models of laboratory equipment using Nanite and Lumen (UE5).Physics-Based Interaction: Realistic handling of tools and materials in a VR space.Safety Protocols: Simulated emergency scenarios that would be too dangerous or costly to perform in real life.Data Logging: Tracks user performance and equipment telemetry for post-session analysis.

Used programs: Unreal Engine 5, LabVIEW
Test helmet: Pico 4
